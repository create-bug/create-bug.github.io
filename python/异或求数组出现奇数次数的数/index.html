异或求数组出现奇数次数的数

要求：

有一数组，如[1,2,3,4,1,2,4],
求其中出现奇数次数的数，且尽量避免创建新变量

异或示例

通过[异或]实现， 例如，在python 中

1^1
0 1^2^3^4^1^2^4 3

程序实现

def xorArray(arr): result = 0 for i in arr: result = result ^ i return result

? 当出现多个0
